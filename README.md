# mipiInk.Software
Repository for the Electronics; schematics/PCB/STEP/etc

# Breadboard prototype
The below is the latest version of the breadboard for the mipiInk project. The aim is that this image will align with the final product, and will be kept up to date.

![Breadboard](/Breadboard/prototype_breadboard.png)

Currently this only covers the following:
- SD Card; utilising SPI1; SCLK (Pin10), MOSI (Pin11), MISO (Pin12), CS (Pin15)
- eInk connector; utilising SPI1; SCLK (Pin10), MOSI (Pin11), MISO (N/A), CS (Pin2), DC (Pin8), Reset (Pin9) and Busy (Pin1)
